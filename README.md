## How to for scripts
### weather data
weather data to csv:

column dropping etc:
weather_data_cleaning.py
coco mapping:
coco_weather_mapping.py
hourly to daily aggregated:
aggregate_hourly_to_daily_weather.py
### trip data (presented only for 2013-2019, other cleaning scripts are modified, because initial data is different)
first cleaning:
data_batch_cleaning.py
second cleaning:
data_batch_cleaning_second_time_plus_report.py
hourly to daily:
data_aggregation.py
### cleaning report
cleaning_report.py
